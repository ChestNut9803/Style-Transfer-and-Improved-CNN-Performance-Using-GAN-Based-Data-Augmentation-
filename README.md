# Style-Transfer-and-Improved-CNN-Performance-Using-GAN-Based-Data-Augmentation-
This project establishes a comprehensive deep learning framework for classifying and generating images of cows and horses. The workflow integrates three advanced neural network architectures: CNNs, GANs, and CycleGANs.
